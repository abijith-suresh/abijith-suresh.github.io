A comprehensive fitness tracking application built with React Native, helping users log workouts, track progress, and stay motivated.

## Features

- **Workout Logging**: Track exercises, sets, reps, and weights
- **Progress Charts**: Visualize strength and endurance improvements
- **Custom Workouts**: Create and save personalized workout routines
- **Social Features**: Share achievements and follow friends
- **Reminders**: Push notifications for scheduled workouts

## Tech Stack

- **Mobile**: React Native, Expo
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Charts**: Victory Native for data visualization

## Development

Built with Expo for rapid cross-platform development, supporting both iOS and Android with a single codebase.
