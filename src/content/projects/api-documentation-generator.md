An automated API documentation generator that creates beautiful, interactive documentation from OpenAPI/Swagger specifications.

## Features

- **Auto-generation**: Parse OpenAPI specs automatically
- **Interactive Examples**: Try API endpoints directly
- **Code Samples**: Generate code in multiple languages
- **Search**: Fast search across all endpoints
- **Themes**: Customizable themes and branding

## Tech Stack

- **Core**: Rust (compiled to WebAssembly)
- **Frontend**: Svelte
- **Parsing**: OpenAPI 3.0 specification parser
- **Deployment**: Static site generation

## Innovation

Used Rust + WebAssembly for parsing large API specs with near-native performance in the browser, reducing generation time by 10x compared to JS-based solutions.
