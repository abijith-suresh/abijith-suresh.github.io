A modern portfolio website built with Astro 5, featuring a clean design philosophy inspired by astro-erudite. The site emphasizes performance, accessibility, and content-first design.

## Features

- **Content Collections**: Type-safe content management with Zod validation
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Fast Performance**: Static site generation with minimal JavaScript
- **Type Safety**: Full TypeScript support throughout the codebase

## Technical Highlights

Built with the latest web technologies, this portfolio demonstrates:

- Astro's Content Layer API for managing projects and blog posts
- shadcn/ui-inspired theming system with CSS custom properties
- Centralized configuration for easy maintenance
- Clean, semantic HTML with proper accessibility considerations

## Future Enhancements

- MDX support for interactive blog posts
- RSS feed for blog subscribers
- View transitions for smooth navigation
- Table of contents for long-form content
