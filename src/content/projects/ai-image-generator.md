An AI-powered image generation platform that transforms text prompts into stunning images using Stable Diffusion models.

## Features

- **Text-to-Image**: Generate images from text descriptions
- **Style Presets**: Various artistic styles and themes
- **Image Editing**: Modify generated images with additional prompts
- **Gallery**: Save and share generated images
- **Batch Generation**: Create multiple variations

## Tech Stack

- **Backend**: Python, Flask, Stable Diffusion
- **Frontend**: React, TailwindCSS
- **ML**: PyTorch, Hugging Face Transformers
- **Storage**: AWS S3 for image storage

## Technical Highlights

Optimized inference pipeline to reduce generation time by 40% and implemented queue system for handling concurrent requests.
