A desktop application for developers to organize, search, and manage code snippets efficiently with powerful search and tagging capabilities.

## Features

- **Snippet Storage**: Save code snippets with descriptions
- **Syntax Highlighting**: Support for 100+ programming languages
- **Search**: Full-text search across snippets
- **Tags & Collections**: Organize with tags and collections
- **Export**: Export snippets to GitHub Gists

## Tech Stack

- **Desktop**: Electron
- **Frontend**: React, TypeScript
- **Database**: SQLite for local storage
- **Editor**: Monaco Editor (VS Code's editor)

## Design Philosophy

Built as a desktop app for privacy and offline access, with a focus on keyboard-driven workflow for developer productivity.
