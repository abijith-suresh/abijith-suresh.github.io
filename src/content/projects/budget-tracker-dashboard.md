A personal finance dashboard that helps users track expenses, set budgets, and gain insights into their spending patterns.

## Features

- **Expense Tracking**: Log expenses with categories
- **Budget Management**: Set monthly budgets per category
- **Visualizations**: Interactive charts and graphs
- **Recurring Transactions**: Automatic logging of subscriptions
- **Export Reports**: Generate PDF reports

## Tech Stack

- **Frontend**: Vue 3, TypeScript
- **Backend**: Firebase (Firestore, Auth)
- **Charts**: Chart.js
- **Styling**: TailwindCSS

## User Experience

Focused on simplicity and speed, allowing users to log expenses in under 5 seconds with smart category suggestions.
